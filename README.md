# jArtnet

This is an implementation of the Art-Net protocol by Artistic Licence Holdings Ltd.

Packets currently implemented:

* ArtPoll
* ArtPollReply
* ArtDmx
* ArtDiagData
* ArtIpProg
* ArtIpProgReply
* ArtSync
* ArtTimeCode
* ArtCommand

Currently tested with:

* ENTTEC PRO-Manager Art-Net Recorder
